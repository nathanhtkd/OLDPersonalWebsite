Projects

PhilSA Points Tracker (PPT)

Currently, I am working on this project with a couple of friends! We got this idea because the secretary of the Phillipine Student Association had to manually input points for 350+ members everytime they went to a club event. This caused eligibility requirement notices to come out later than expected. So, our plan is to get the points straight from Google Forms and input them automatically to Google Sheets. To implement this project, we are using Google Apps Script, Javascript, HTML, and CSS.

Space Invaders (SI)

This was a project I worked on over Summer 2022. Space Invaders is a popular shoot 'em up arcade game. It has very simple UI with pretty basic features. One of my favorite parts of working on this project was using the Pygame library. Pygame is a library with modules that help support the game design process. For example, it made calculating for pixel collisions really simple with the mask function, and allowed me to draw sprites on the screen with ease. Lastly, I employed Pyinstaller to bundle my Python application and all of its dependencies into a single package for Windows and Linux operating systems. To implement this project, I used Python and Pygame.

React Apps (RA)

During the Spring semester of 2022, I joined a learning based project team called React Apps. This group was formed from members of the Aggie Coding Club. During this time, we met every week and learned how to code with HTML, CSS, and Javascript through CodeAcademy. To solidify our knowledge, we created simple websites with the respective languages. To build teamwork, we engaged in discussions about how our week went and our career goals. This promoted camaraderie within the team and motivated us to keep out with the material assigned to us. Sadly, we couldn't dive deep into React as intended, but I had a great time learning with this team!
